## 音乐

### 播放音乐
播放指定的背景音乐。
```
@[play_bgm] {
    "path": "音频路径",
    "db": 0,
    "pan": 0
}
```
**参数说明**：
- `path`：音频文件路径（参考：[隐藏音频路径索引](../../special/隐藏音频路径索引.md)）。
- `db`：音量大小，可选，默认值为 0。
- `pan`：声道平衡，负数为左声道，正数为右声道，可选，默认值为 0。

### 改变音乐
切换当前播放的背景音乐。
```
@[change_bgm] {
    "has_bgm": false,
    "path": "音频路径"
}
```
**参数说明**：
- `has_bgm`：是否有背景音乐，设为 `false` 可停止当前音乐。
- `path`：音频文件路径，可选（参考：[隐藏音频路径索引](../../special/隐藏音频路径索引.md)）。

### 音乐淡入
音乐淡入播放。
```
@[fade_in_bgm] 1
```
参数为淡入持续时间（秒）。

### 音乐淡出
音乐淡出停止。
```
@[fade_out_bgm] 1
```
参数为淡出持续时间（秒）。

### 停止音乐
停止当前播放的背景音乐。
```
@[stop_bgm]
```
