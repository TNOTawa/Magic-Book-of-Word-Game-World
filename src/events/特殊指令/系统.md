## 系统

### 返回标题界面
直接退出当前流程并返回标题界面。

```
@[return_to_title_screen]
```

### 直接退出游戏
快速关闭游戏流程。

```
@[quit_game]
```

### 设置成就完成
将指定成就状态设为已完成。

```
@[set_achievement]
```

### 清除成就完成状态
将指定成就状态清除为未完成。

```
@[clear_achievement]
```
### 终止
一条指令让游戏陷入无限等待，至少还可以从右上角X退出

```
@[ending]
```

### 无条件外部连结
无条件打开url，不需要用户确认

```
@[open_url] ""
```

### 切换世界
从当前世界直接跳到另一个世界里去，中间自动淡入，参数是世界id<br>
**世界id可以在文件目录"Word Game World\resources\resources\bundled-worlds-index.json"找到**

id格式通常是99cc3e6d-8412-4768-b430-213e67aa4755这么一串uuid

```
@[editor_world_transport] ""
```
