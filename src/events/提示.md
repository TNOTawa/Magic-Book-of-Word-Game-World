# 提示

*该页面展示的黑魔法均为在游戏内插入后的默认参数*

## 叮！
```

@[event_end_hint]

```

## 物件呼吸灯
```
@[add_loop_light] {
  
    "target": null
  
}
```

## 停止物件呼吸灯
```
@[remove_loop_light] {
  
    "target": null
  
}
```

## 强调位置
```
@[highlight] {
  
    "pos": [
      0,
      0
    ],
    "text_count": 1,
    "is_h": true
  
}
```

## 改变选单提示
```
@[set_editor_project_info] {
  
    "title": "",
    "description": "",
    "hint_1": "",
    "hint_2": "",
    "hint_3": ""
  
}
```
