# 条件

*该页面展示的黑魔法均为在游戏内插入后的默认参数*

## 符合时执行
```
@[if] {
  
    "type": "switch",
    "varOption": {
      "varId": null,
      "operation": "==",
      "varValue": 0
    },
    "switchOption": {
      "varId": null,
      "operation": "==",
      "varValue": false
    }
  
}
```

## 否则执行

```
@[else]
```

## 结束条件
```
@[end_if]
```
