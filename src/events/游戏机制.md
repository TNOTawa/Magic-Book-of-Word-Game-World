# 游戏机制

*该页面展示的黑魔法均为在游戏内插入后的默认参数*

## 宣告拆字规则
```
@[append_split_rule] {
  
    "from": [
      null,
      null
    ],
    "to": null
  
}
```

## 宜告组字规则
```
@[append_merge_rule] {
  
    "from": [
      null,
      null
    ],
    "to": null
  
}
```

## 宣告成句规则
```
@[append_sentence_rule] {
  
    "text": "成句",
    "has_hint": false,
    "has_animation": true,
    "memory": [],
    "progress": 0,
    "level": 5
  
}
```

## 取消成句规则
```
@[delete_sentence_rule] {
  
    "text": "成句"
  
}
```
